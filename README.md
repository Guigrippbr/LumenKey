# LumenKey
LumenKey é um gerenciador de senhas local, focado em segurança e privacidade. Com uma interface intuitiva, o app permite que os usuários armazenem suas credenciais de forma criptografada, sem depender de nuvem.

**-> Tecnologias utilizadas:**

• Flutter para desenvolvimento cross-platform.

• SQLite para armazenamento local seguro.

• Criptografia avançada para proteger as senhas.

**Objetivo:** Oferecer uma solução simples e eficiente para quem busca controlar suas senhas de forma privada, com foco na experiência do usuário.
